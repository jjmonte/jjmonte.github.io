Reg Rotator
A web application designed for college radio stations to manage rotation, browse artists, and (soon) to create playlists. This CRUD app uses a MERN stack (MySQL, Express.js, React.js, Node.js) to allow users to add albums to a database and categorize them by play priority, genre, release date, and more. This project was initially meant to digitalize a primarily written process, but is growing into a hub for various radio tasks. Collaboration with Victor Perez.
Air Track
This web application was originally a training project I planned and developed while learning AngularJS. The project evolved from a simple HTML/AngularJS app into a single page React app. Utilizing Cesium.js for maps and the OpenSky Network API, Air Track allows users to browse airports by state (US only) and view the flights arriving and departing from the specified airport over a selected time span.
